# ComfyUI Storyboard

This is a ComfyUI custom node that provides a storyboard creation interface.

## Installation

1. Copy this folder (`comfyui-storyboard`) to your `ComfyUI/custom_nodes/` directory.
2. Restart ComfyUI.

## Usage

1. Open ComfyUI.
2. You will see a "Storyboard" button in the menu.
3. Click it to open the Storyboard interface.
4. The current workflow in ComfyUI will be automatically loaded into the Storyboard.
